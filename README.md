# Sistema de detección de intrusiones casero

Este proyecto contiene todos los ficheros .ino necesarios para crear una alarma casera con dos Arduino UNO.

Los archivos ardu1.ino y ardu2.ino corresponden con el código que se utilizó en la herramienta Tinkercad para simular el proyecto.

Los archivos ardu1-fisico.ino y ardu2-fisico.ino son los archivos finales utilizados en el montaje físico de los dos arduinos. 

Se puede encontrar toda la información relativa al proyecto en el fichero memoria-proyecto.pdf
